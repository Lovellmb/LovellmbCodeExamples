Forecast For You

Welcome to our Website

This website was Created by Max Lovell and Owen Ensly as a part of a web apps project for CSE 383. The dashboard allows users to obtain weather data for the next week by entering a desired zip code to check. The weather data is fetched from the OpenMeteo and Weather.Gov API's, and is then summarized using openAI. The Creator pages give brief introductions to the creators and explain what they brought to this project. The History page allows users to review all previous results and search by a desired date or zip code.

Get an Accurate Forcast with Data from Two Sources

With our website you can find the weekly forcast for any provided zipcode. Our dashboard will display the max temperature, windspeed, and percipitation chance from not just one but 2 weather APIs. Furthermore, at the bottom of each search you will find a newly generated AI summary of the data providing all of the info in an easier to read format.
